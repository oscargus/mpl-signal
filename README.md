# mplsignal

Matplotlib extension for signal processing

Documentation can be found [here](https://mplsignal.readthedocs.io/).

## Installation

You can install using `pip`:

```bash
pip install mplsignal
```

## Development Installation


```bash
pip install -e ".[dev]"
```
